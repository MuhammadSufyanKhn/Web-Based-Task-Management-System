# Web-Based Task Management System

## 📌 Project Description
The **Web-Based Task Management System** is a web application that helps users manage their daily tasks efficiently.  
Users can create, update, delete, and track tasks through a simple and user-friendly interface.  
This project demonstrates the implementation of CRUD operations, authentication, and database integration in a web-based environment.

## 🎯 Objectives

- To develop a web-based task management system
- To manage tasks efficiently
- To demonstrate full-stack web development concepts
- To provide a user-friendly interface
- To improve productivity and organization

## ✨ Features

- User Registration and Login
- Add New Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Task Status Management
- Secure Authentication
- Responsive User Interface

## 🛠️ Tech Stack

**Backend:**
- **Language:** C#
- **Framework:** ASP.NET Core Web API
- **Data Access:** Entity Framework Core 
- **Database:** SQL Server / SQLite

**Frontend:**
- **Languages:** ReactJS

## 📂 Project Structure

- `TaskManagementAPI/`: Contains the server-side logic, controllers, and database configurations.
- `TaskManagerUi/`: Contains the client-side application and styling.
- `package.json`: Manages frontend dependencies and build scripts.
